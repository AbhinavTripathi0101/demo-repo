# Simple Calculator

This is a simple Python calculator that supports addition, subtraction, multiplication, and division.

## Usage

1. Run the calculator:

```bash
python3 calculator.py
```

2. Follow the prompts to select an operation and enter numbers.

- Supported operations:
  - 1: Add
  - 2: Subtract
  - 3: Multiply
  - 4: Divide

If you enter an invalid choice, the program will notify you.

## Example

```
Simple Calculator
Select operation:
1. Add
2. Subtract
3. Multiply
4. Divide
Enter choice (1/2/3/4): 1
Enter first number: 5
Enter second number: 3
Result: 8.0
```
